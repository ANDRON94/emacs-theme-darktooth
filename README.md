# Darktooth

An Emacs theme, *From the darkness... it watches.*

## Installation

Darktooth can be easily installe via MELPA

[Read this for MELPA setup details](http://melpa.org/#/getting-started)

Once setup, use `M-x package-install darktooth-theme`

![](darktooth-vera.png)
