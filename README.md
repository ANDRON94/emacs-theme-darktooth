# Darktooth

A remix of the excellent theme Gruvbox.

# Modes supported

- Whitespace
- Rainbow delimiters
- Linum relative
- Highlight indentation
- Smartparens
- Elscreen
- Ag
- Diff/EDiff
- Js2
- Org
- Powerline
- Popup
- Pos-tip
- Anzu
- Company
- Helm
- Term / Ansi Colors
